Three major areas in Execution
===============================
1.Stack area
2.Class static Area
3.Heap area

stack area : execution happens in stack area
first : class loader-> loads all static members of class into Class static area
by creating different static pool areas for different classes.

second : main method in to the stack and starts execution.

third : garbage collector -> clears the jvm memory.

Heap area : whenever we create objects, new key-word comes into action.


three operations of new keyword
===============================
1.creating object in heap area
2.loading all non-static class members into the object.
3.returns the object address to object reference variable.
