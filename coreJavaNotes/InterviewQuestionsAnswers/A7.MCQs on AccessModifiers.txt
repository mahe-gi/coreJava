a few more questions
=======================
Q1. What is the default access modifier for a class member if no modifier is specified?
a) public
b) private
c) default (package-private)
d) protected

Answer:c) default (package-private)

Q2. Which access modifier allows members to be accessible only within the same class?
a) public
b) private
c) default
d) protected

Answer:b) private

Q3. Which access modifier allows a class member to be accessible within the same package and in subclasses of other packages?
a) public
b) private
c) default
d) protected

Answer:d) protected

Q4. Which modifier(s) can be applied to top-level classes?
a) public only
b) default only
c) private and protected
d) public and default

Answer: d) public and default

Q5. Which of the following is true about private access modifier?
a) Private members are accessible within the same package.
b) Private members are accessible in subclasses.
c) Private members are accessible only within the same class.
d) Private modifier can be used with top-level classes.

Answer: c) Private members are accessible only within the same class.

Q6. What will happen if you declare a top-level class as protected?
a) It will compile successfully.
b) It will compile but throw runtime error.
c) Compilation error.
d) It will work like default access.

Answer: c) Compilation error.

Q7. If a class is declared with no access modifier, where can it be accessed?
a) Anywhere
b) Only in its own class
c) Within its own package
d) Only in subclasses

Answer: c) Within its own package

Q8. Which of the following statements is correct about protected members?
a) They are accessible only within the same class.
b) They are accessible within the same package and in subclasses outside the package.
c) They are accessible only within subclasses.
d) They are accessible everywhere like public members.

Answer: b) They are accessible within the same package and in subclasses outside the package.

Q9. Which of these modifiers cannot be used for local variables?
a) public
b) private
c) protected
d) All of the above

Answer: d) All of the above
(Access modifiers are not applicable to local variables)

Q10. What is the output of below code?
class A {
    private int x = 10;
}

class B {
    public static void main(String[] args) {
        A obj = new A();
        System.out.println(obj.x);
    }
}
a) 10
b) Compilation error
c) Runtime error
d) 0

Answer:b) Compilation error
(private member x not accessible outside class A)
